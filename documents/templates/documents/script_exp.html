Network Traffic Capture Code

[Instructions for Use]

1. Scapy에서는 WinPcap을 사용하여 패킷 스니핑 및 송신을 지원합니다. WinPcap을 다운 받으세요.

   WinPcap 다운로드 링크: www.winpcap.org/install/default.htm

2. 시그니처에서 제공하는 코드를 다운로드합니다.

3. 이 코드에서는 scapy 라이브러리를 사용합니다. scapy 라이브러리를 설치하지 않았다면, 다음 명령어를 실행하세요.

Terminal
                pip install scapy

4. 코드를 실행하기 전, capture_time 변수를 수정하여 캡처할 시간을 조정할 수 있습니다.

Python
               capture_time = 600  # 기본값: 600초(10분)

5. 코드를 실행하면, 캡처된 데이터는 현재 디렉토리의 "network_traffic.csv" 파일로 저장됩니다.

6. CSV 파일을 업로드합니다.
