시그니처는 네트워크 트래픽 데이터를 수집하는 코드를 제공합니다.

Network Traffic Capture Code

[Instructions for Use]

1. Scapy에서는 WinPcap을 사용하여 패킷 스니핑 및 송신을 지원합니다. WinPcap을 다운 받으세요.

WinPcap 다운로드 링크: www.winpcap.org/install/default.htm

2. 시그니처에서 제공하는 실행파일을 다운로드합니다.

3. network_capture_widget.exe 을 더블 클릭하여 실행하세요.

4. Capture Time (s) 에 시간(초)을 입력하고 Start 버튼을 누르면 데이터 수집이 시작됩니다.

5. 데이터 수집이 시작되면 아래와 같은 문구와 진행되고 있는 퍼센트(%)가 보이게 됩니다.

6. 데이터 수집이 완료되면 End 버튼을 눌러 실행 파일을 종료하면 됩니다.

7. 수집된 네트워크 데이터는 실행 파일이 있는 디렉토리에 .csv 파일과 .pcap 파일로 저장됩니다.

8. 네트워크 데이터를 수집을 완료했다면 사이트에 업로드 하여 데이터를 분석해보세요.
